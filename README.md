# Excel-Project
📊 E-Commerce Sales Dashboard
An interactive **E-Commerce Sales Dashboard** built using **Excel Pivot Table** to analyze customer purchasing behavior, sales performance, product trends, shipping efficiency, and customer satisfaction.

## 📌 Overview

This dashboard provides insights into:

- Total Orders
- Total Quantity Sold
- Total Revenue
- Customer Ratings
- Average Delivery Time
- Weekly Sales Trends
- Sales by Purchase Source
- Product Performance by Gender
- Geographic Order Distribution
- Shipping Time Analysis
- Customer Satisfaction Ratings

## 🚀 Features

- 📈 **Sales Trend Analysis**
  - Last 13-week sales and revenue trend.

- 🛍️ **Purchase Source Analysis**
  - Compare purchases from:
    - App
    - Instagram
    - Partner App
    - Target.com
    - Website

- 👥 **Gender-Based Analysis**
  - Product purchases by gender.
  - Source-wise customer distribution.

- 📦 **Quantity Distribution**
  - Order quantity frequency.

- 🗺️ **Location Analysis**
  - State-wise order quantity across California.

- 🚚 **Shipping Performance**
  - Delivery time distribution.

- ⭐ **Customer Satisfaction**
  - Monthly rating comparison (1–5 Stars).

- 🎯 **Interactive Filters**
  - Order Source
  - Gender
  - Customer Rating

## 📈 Dashboard Insights

- Website and App generate the highest number of orders.
- Female customers contribute the largest share of purchases.
- T-Shirts and Jeans are the most purchased products.
- Most orders are delivered within 2–3 days.
- Revenue remains relatively stable with periodic peaks.
- Customer satisfaction is primarily concentrated around 4-star ratings.
